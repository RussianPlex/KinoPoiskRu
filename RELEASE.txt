Release process:
===============================================================================
 1) make sure logging entries are removed in Info.plist config configuration files;
 2) run 'ant dist' (or an individual plugin dist target);
 3) upload file (-s) to sourceforge;
 4) update Plex wiki pages if necessary:
   - http://wiki.plexapp.com/index.php/WikipediaRu
   - http://wiki.plexapp.com/index.php/KinoPoiskRu
   - http://wiki.plexapp.com/index.php/1TvRu
   - http://wiki.plexapp.com/index.php/WeeWzaTvRu
 5) make a release announcement.


KinoPoiskRu
===============================================================================

CURRENT TARGET

KinoPoiskRu-1.2
- Улучшение получения картинок
  * Добавление большого русско-язычного ярлыка (постера) в случае когда он есть;
  * Изменение алгоритма выбора постеров с целью убрать не-рускоязычные картинки;
  * Улучшение алгоритма выбора картинок с целью не использовать картинки с плохим
    разрешением и неподходящей ориентацией (для постеров - предпочтение отдается
    вертикальной, для фотографий - горизонтальной ориентации);
- Улучшение определения выбора результатов:
  * использование оригинального названия фильма сравнения;
  * улучшение алгоритма сравнения года выпуска фильма;
- Устранение ошибок;
  * Устранение ошибки сортировки картинок;
  * устранение ошибки парсинга актеров (спасибо moddom);
  * устранение ошибки парсинга рэйтинга (спасибо moddom);
  * устранение ошибки выбора фильма в случае когда фильм найден сразу (спасибо moddom);
  * устранение ошибки выбора фильма в случае когда результат не имеет года;
- Улучшение конфиругации плагина
  * Упрощение конфигурации (удаление конфигурации логов);
  * Изменение конфигурации плагина больше не требует перезагрузки сервера;
  * Уменьшение количества картинок скачиваемых по умолчанию для увеличения скорости
    работы уменьшения интернет трафика;
- Добавление поддержки IMDB. Теперь по умолчанию, включена поддержка IMDB, что
  позволяет использовать плагин KinoPoiskRu c другими плагинами требующими идентификатор
  IMDB, например MoviePosterDB.
- Прочие изменения
  * Добавление парсинга стран;
  * Переход на использование логов Plex;

Если нет желания видеть неруские постеры (ярлыки), в установках плагина,
для максимального количества постеров следует выбрать значение 1. Если эта
установка будет более 1 - в постеры будут попадать не русско-язычные картинки.

ВНИМАНИЕ! Замечен странный баг ПМС где установки плагина не загружаются в окне
"Metadata Agent Settings" - вы заметите что установки плагина будут все время
загружаться одни и те же. В этом случае, сохранение установок все равно возможно,
только следует убедиться в том что прежде как установки будут сохранены, каждая
строка в окне установок плагина будет иметь должное значение.
